SAÉ 2.04

Contexte

Cette SAÉ était à réaliser dans le cadre de la ressource de base de données par équipe de 4, sous la direction
de notre professeur de TP, Mme Julie Rossignol, et de façon plus générale par Mme Anaïs Durand.
Nous avions environ 1 mois et demi (commençant vers le 20 Mars et finissant le 4 Mai).

Objectif

L'objectif de cette SAÉ était de prendre un jeu de données sur internet, d'en faire une base de données, puis de faire des statistiques
avec les données recueillies. Nous devions effectuer les requêtes de la base de données en SQL et nous devions réaliser les statistiques en Python.

Travail Réalisé

Pour ma part, j'ai utilisé plusieurs outils pour pouvoir réaliser cette SAÉ : J'utilisais pgAdmin et postre SQL pour faire tourner
la base de données ainsi que Pycharm de JetBrain pour le code Python (les requêtes SQL s'effectuaient à l'intérieur du code Python).
Nous avons utilisé Git pour permettre le travail d'équipe et le partage du code. Dans un premier temps, nous avons choisi un jeu de données
qui répertoriait les accidents de la route en France entre 2005 et 2022. Pour pouvoir voir l'évlution des données nous avons choisi de prendre
les données des années 2005, 2013 et 2018 (les données post 2018 n'étant pas utilisables). Ensuite nous avons trié les données afin de garder
les plus "propres" et nous les avons mis dans la base de données. Une fois ceci fait, nous avons fait plusieurs requêtes à la base de données
afin de pouvoir faire les différents graphiques et histogrammes. (Voir exemples ci-dessous)
